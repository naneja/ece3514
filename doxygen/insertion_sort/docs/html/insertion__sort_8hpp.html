template<typename T>
void insertion_sort ( T * data,
int n )

Sorts an array using the insertion sort algorithm.

Reorders the elements in the range [data, data + n) into non-decreasing order using insertion sort.

Template Parameters
TType of elements stored in the array.
Parameters
dataPointer to the first element of the array.
nNumber of elements in the array.
Precondition
data must point to a valid array of at least n elements.
Type T must support:
  • Move assignment
  • Copy construction
  • operator< comparison

@complexity Worst-case: O(n^2) comparisons and moves Best-case: O(n) (already sorted input)

Note
Stable sorting algorithm.
Warning
Behavior is undefined if data is null and n > 0.
