* In this lab, you will learn how to:

  * Use nested loops to traverse rows and columns
  * Print a grid (board) format using std::cout
  * Understand how multi-dimensional arrays are stored and accessed

## Background

* A 2D array in C++ is like a table with rows and columns. It is declared using:

```cpp
dataType arrayName[rows][columns];
```

* You can access each element using:

```cpp
arrayName[row][column]
```

* To print all elements, you will use nested loops.
